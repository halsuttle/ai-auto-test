## 🔧 管理命令

### 添加知识
```
指令：添加知识到知识库
格式：
- 分类：[技术知识 | 工作流程 | 业务信息 | 经验教训 | 外部导入]
- 标题：[知识标题]
- 内容：[知识内容或文件路径]
- 标签：[可选标签，用于检索]
```

### 检索知识
```
指令：从知识库检索 [关键词]
返回：匹配的知识条目列表
```

### 更新知识
```
指令：更新知识库中的 [知识标题]
内容：[新的内容]
```

### 导出知识
```
指令：导出知识库 [分类/全部] 为 [格式：md/docx/pdf]
```

## 📊 知识入库流程

1. **接收内容** - 从用户、网页、文档等来源获取
2. **分类整理** - 判断归属类别
3. **格式化** - 统一 Markdown 格式，添加元数据
4. **存储归档** - 保存到对应文件夹
5. **索引更新** - 更新 `INDEX.md` 索引文件
6. **确认反馈** - 向用户确认入库成功
